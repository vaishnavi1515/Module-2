# Ex 2c:Lambda Function : Lambda function to multiply a number by 45

## Aim
To write a Python lambda function that takes z as a parameter and returns z Ã— 45.
## Algorithm
1.Start the program.

2.Create a lambda function that takes one parameter z and returns z * 45.

3.Read a value for z from the user.

4.Call the lambda function with the input value.

5.Display the result.

6.End the program.
## Program
```
multiply_by_45=lambda z: z*45
z=int(input())
result=multiply_by_45(z)
print(result)
```
